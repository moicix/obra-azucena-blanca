# Proyecto de Obra de Pascua: "La Azucena Blanca"

Este repositorio contiene la estructura de directorios y los documentos iniciales para la puesta en escena de la obra de teatro "La Azucena Blanca", adaptada para el grupo de jóvenes del Movimiento Familiar Cristiano para la temporada de Pascua.

El proyecto está diseñado para ser altruista, sin presupuesto, y flexible para acomodar la participación de jóvenes entre 12 y 24 años con posibles ausencias.

## Descripción de Secciones

### **00_PROYECTO_Y_PLANIFICACION**
Contiene los documentos generales del proyecto, el calendario de hitos y la información de contacto de todos los participantes. Es el punto de partida para entender el alcance y las bases del proyecto.

### **01_GUIÓN_Y_ADAPTACION**
Aquí se encuentra el guion principal de la obra, así como un desglose de personajes y escenas fundamental para la planificación de ensayos. También se incluye el cuento original y un espacio para versiones anteriores del guion.

### **02_PRODUCCION**
Esta sección se enfoca en la ejecución. Incluye calendarios de ensayos detallados, un sistema para registrar notas de cada ensayo y, crucialmente, un plan de sustitutos (understudies) para manejar las posibles ausencias del elenco. También cubre aspectos técnicos básicos como iluminación y sonido, adaptados a un entorno sin presupuesto.

### **03_ELENCO_Y_EQUIPO**
Gestiona toda la información relacionada con el talento humano: lista de actores, sus roles y un registro de su disponibilidad. Las "Fichas de Personaje" permitirán a cada actor profundizar en su papel.

### **04_DISEÑO_Y_RECURSOS_(SIN_PRESUPUESTO)**
Dada la naturaleza altruista del proyecto, esta sección está dedicada a la creatividad y la reutilización. Contiene ideas para vestuario y utilería que se pueden conseguir con recursos mínimos (donaciones, objetos caseros, DIY) y un espacio para inspiración visual.

### **05_COMUNICACION_Y_DIFUSION**
Guía para comunicar y difundir la obra entre la comunidad, utilizando canales gratuitos y efectivos para asegurar una buena asistencia.

### **99_ARCHIVOS_FINALES_Y_FOTOS**
Espacio para guardar la versión final del guion, grabaciones, fotos del evento y cualquier otro material conmemorativo.
